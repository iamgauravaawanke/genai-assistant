# Smart AI Assistant

A Full-Stack AI Assistant built using:

- React.js
- FastAPI
- PostgreSQL
- ChromaDB
- Qwen LLM

## Features

- Chat History
- Multi Sessions
- Memory
- Weather Tool
- Search Tool
- Calculator Tool
- Travel Tool
- PDF Upload
- Retrieval-Augmented Generation (RAG)

## Tech Stack

Frontend:
- React
- Tailwind CSS

Backend:
- FastAPI
- Python

Database:
- PostgreSQL
- ChromaDB
